## Información
Es una librería de [[C++]] para el manejo de [[Arrays]] de forma similar a [[Matlab]]

## Instalación
1. Descomprimir el archivo
2. Instalar `cmake`
3. Correr `cmake .`
4. Correr `sudo make install`

## Compilación de programas 
Usar el comando 
`g++ prog.cpp -o prog -std=c++11 -O2 -larmadillo`
